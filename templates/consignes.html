🛠 Déploiement via GitHub Actions

Pour automatiser la mise à jour sur PythonAnywhere, configurez les secrets suivants dans votre dépôt :

PA_USERNAME - Votre user PA_TOKEN - Token API (Account -> API Token) PA_TARGET_DIR - /home/votreuser/votreprojet PA_WEBAPP_DOMAIN - votreuser.pythonanywhere.com
